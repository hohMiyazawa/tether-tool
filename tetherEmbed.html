Incomplete embedding example

In <head>, you need to include the scripts "orbit.js" and "tetherTool.js":

<script src="js/orbit.js"></script>
<script src="js/tetherTool.js"></script>

In the markup, you need a DOMnode where you want the tether tool inserted:

<div id="insertItHere"></div>

Then, you can call the function "tetherTool" to initialise the tool:

<script>
    tetherTool(document.getElementById("insertItHere"))
</script>
